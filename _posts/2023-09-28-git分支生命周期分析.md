> 团队多，团队开发的微服务多，代码仓多，商业交付场景多，研发组织模式各有特色
> 通过对代码仓及分支的表面现象分析试图能找到一些主要矛盾。

# 技术实现

- gitk：在git command中执行`gitk`就可以使用git的GUI工具查看相关commit信息。
- git merge-base branch_A branch_B：可以查看两个分支之间最近的公共commit。
